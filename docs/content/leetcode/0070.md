### 题目
#### 爬楼梯
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。  
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？  
注意：给定 n 是一个正整数。
#### 样例
输入： 2  
输出： 2
### 题目大意
走楼梯
### 解题思路
#### C++思路一
f[n]=f[n-1]+f[n-2];
### 代码
#### C++思路一
```C++
class Solution {
public:
    int climbStairs(int n) {
        long long f[10000]={0,1,2};
        for(int i=3;i<=n;++i){
            f[i]=f[i-1]+f[i-2];
        }
        return f[n];
    }
};
```
